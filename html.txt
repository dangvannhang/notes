* Những điều cần chú ý trong html

* Hướng dẫn sử dụng Emmet để code html nhanh:
- tag.class_name: dùng để tạo ra một tag nào đó với class_name -> section.section_about -> <section class="section_about></section>
- 

* Sự khác nhau về kiểu dữ liệu của các attributes về thời gian trong input tag:
- type = "time" -> nó sẽ chỉ nhận về giờ, phút và am hoặc là pm thôi.
- type = "date" -> nó sẽ nhận về ngày tháng năm.

* Select rồi option mà muốn có cái cái gọi là option group ở trước các cái option thì dùng <optgroup label="Hiển thị tên group"></optgroup>

* Clear hết data trên form chỉ dùng html thì <input type="reset" value=""> -> Khi mà người dùng click vào thì nó sẽ tự động clear hết tất cả các thông tin của form.

* Có thể xác thực dữ liệu tại các ô input bằng cách thêm attribute là pattern và sau đó là các regex data.
